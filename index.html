In this project you continue to add an embedded style sheet ,starting with some ready-made HTML.Your challengge is to write embedded CSS code to solve the problam set out in items 1-5 below.

Challenge Two Tasks

The CSS Challenge continues with exercises to test your knowledge of working with colour

If you get stuck,refer back to chapter 13 in your book.

  1. Write a rule to set the background color of the page using either RGB or hexadecimal color value , set the page width to less than 100%,and set the page maegins to auto.

  2. Update all color properties with color names to use RGB or hexadecimal value in all existing style rules
  3. Write rules for the five link states or they are a diffrent color than the rest of the text , and when yoe mouse over them , the underline either appears or disappears , The style for the link and hover states should be visually diffrent, and remember that order matters.
  4. Write a rule to apply an appropriate background image to the page header element . Remember that header element height is determined by height of the header, exposing more area of background image .
  5. Modify all the style rules to use shorthand wherever approprirate.
